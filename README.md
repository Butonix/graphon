# Graphon

Engine for rendering charts on native JS.

Features 

- Line charts
- Area charts
- Bar charts
- 2 OY axis chart
- Minimap dragging
- Scaling
- Scrolling
- Night Theme

# Install

```
yarn add graphon
```
